# UML---Modelagem-de-software-do-in-cio-ao-fim_Udemy
Curso introdutório de Engenharia de Software e UML, ideal para iniciantes. Aprenda modelagem de software, orientação a objetos, análise de requisitos e criação de diagramas UML usando a ferramenta ASTAH Community. Sem pré-requisitos, o curso prepara desenvolvedores, analistas e interessados para documentar e planejar softwares de forma prática.
